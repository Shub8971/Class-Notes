VI EDITOR

famous editor in linux has 2 modes of operation
1. command mode
2. insert mode

1. command mode ---- to save, quit and run some command mode operations  -- press esc to switch
2. insert mode- ----- to edit the file   -- press i to switch

some command mode operations

:wq, :wq!, :x, :x! ---- save and quit and !means forcefully

:q, :q! ---- only quit
:w -- only save
x --- delete single char
r --- replace char
dd --- delete line and long press d deletes everything
shift+h --- top of file
shift+g --- bottom of the file
o ---- creates new line below and we come in insert mode
/pattern --- to search a pattern
