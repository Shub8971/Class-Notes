*********Kubernetes Cluster on AWS EC2 (Amazon Linux 2) using kubeadm*************

Architecture:

- 1 Control Plane (Master)

- 1+ Worker Nodes

- OS: Amazon Linux 2

- Runtime: containerd

- CNI: Calico
---------------------------------------------------------------------------
1) AWS EC2 Prerequisites

Instance Requirements
Node	Type	CPU/RAM
Master	t3.medium	2 vCPU / 4 GB
Worker	t3.medium	2 vCPU / 4 GB
------------------------------------------------------------------------------
2) 
2.1 Disable Swap (MANDATORY)

Amazon Linux usually has no swap, but verify:

#free -h


If swap exists:

#sudo swapoff -a
--------------------
2.2 Enable kernel modules

cat <<EOF | sudo tee /etc/modules-load.d/k8s.conf
overlay
br_netfilter
EOF

#sudo modprobe overlay
#sudo modprobe br_netfilter

--------------------
2.3 Sysctl settings

cat <<EOF | sudo tee /etc/sysctl.d/k8s.conf
net.bridge.bridge-nf-call-iptables  = 1
net.bridge.bridge-nf-call-ip6tables = 1
net.ipv4.ip_forward                 = 1
EOF

#sudo sysctl --system

------------------------------------------------------------------------------
3️] Install Container Runtime (containerd)
sudo yum update -y
sudo yum install -y containerd

***Configure containerd:

sudo mkdir -p /etc/containerd
containerd config default | sudo tee /etc/containerd/config.toml

***Edit:

sudo vi /etc/containerd/config.toml

***Change:

SystemdCgroup = true

***Restart:

sudo systemctl restart containerd
sudo systemctl enable containerd

--------------------------------------------------------------------------------
4] Install kubeadm, kubelet, kubectl (ALL nodes)

****Add Kubernetes repo:

cat <<EOF | sudo tee /etc/yum.repos.d/kubernetes.repo
[kubernetes]
name=Kubernetes
baseurl=https://pkgs.k8s.io/core:/stable:/v1.29/rpm/
enabled=1
gpgcheck=1
gpgkey=https://pkgs.k8s.io/core:/stable:/v1.29/rpm/repodata/repomd.xml.key
EOF

----------------------------------------------------------------------------
Install packages

sudo yum install -y kubelet kubeadm kubectl

Enable kubelet:

sudo systemctl enable kubelet

---------------------------------------------------------------------------

5] Configure CRI for kubelet (IMPORTANT on Amazon Linux)

# sudo mkdir -p /etc/systemd/system/kubelet.service.d
--------------------------------------------------------------------------------

cat <<EOF | sudo tee /etc/systemd/system/kubelet.service.d/0-containerd.conf
[Service]
Environment="KUBELET_EXTRA_ARGS=--container-runtime-endpoint=unix:///run/containerd/containerd.sock"
EOF

-----------------------------------------------------------------------------------

sudo systemctl daemon-reload
sudo systemctl restart kubelet

------------------------------------------------------------------------------------

6] Initialize Kubernetes Master (ONLY on MASTER)

---------------------------------------------------------------
sudo kubeadm init \
  --pod-network-cidr=192.168.0.0/16 \
  --cri-socket=unix:///run/containerd/containerd.sock
  
---------------------------------------------------------------
⚠️ SAVE the kubeadm join command
---------------------------------------------------------------

******Configure kubectl access

mkdir -p $HOME/.kube
sudo cp /etc/kubernetes/admin.conf $HOME/.kube/config
sudo chown $(id -u):$(id -g) $HOME/.kube/config

-----------------------------------------------------------------
Test:

kubectl get nodes

--------------------------------------------------------------------------------------------------
7] Install Calico Network Plugin

kubectl apply -f https://raw.githubusercontent.com/projectcalico/calico/v3.26.1/manifests/calico.yaml

----------------------------------------------------------------------------------------------------------
Check:

kubectl get pods -n kube-system

----------------------------------------------------------------

8] Join Worker Nodes

Run on each worker node:

sudo kubeadm join <MASTER_PRIVATE_IP>:6443 \
  --token <token> \
  --discovery-token-ca-cert-hash sha256:<hash> \
  --cri-socket=unix:///run/containerd/containerd.sock

----------------------------------------------------------------------------------------------------

Verify on master:

kubectl get nodes
